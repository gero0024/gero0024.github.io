Tag list

Paragraph
The paragrapgh (p) tag is a self closing tag that allows you to write simple paragrapghs within them.
Header
The Header (h1) tag is used for adding headings to the website. The tag goes from h1 to h6 and decrease in size the bigger the number.
Strong
The Strong (strong) tag is used for making text bold, and this is used to bring attention to important text.
Emphasis
The Emphasis (em) tag makes the text italicized, so this is good for bringing attetion without commiting to bolding the text.
Abbreviation
The Abbreviation (abbr) tag allows you to save space by abbreviating text. This would invlove things like organization names such as NBA.
